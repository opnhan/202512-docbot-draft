# 병원 매출 데이터 스키마

## 파일 정보
- **파일명**: 해외 매출액 매칭 통합본_v4 utf.csv
- **설명**: 병원 1곳의 월별 매출 및 요일별 매출 비중
- **행 단위(Granularity)**: 병원 1개 + YearMonth 1개 + 국가 1개 = 1 row
- **PK(기본키) 후보**: (YearMonth, 코드, 국가) 조합

## 컬럼 정의

### 1. YearMonth
- **타입**: string / int[6]
- **설명**: 매출 기준 연월 (YYYYMM)
- **예시**: 202101

### 2. 국가
- **타입**: string
- **설명**: 매출/환자와 관련된 국가 정보 (환자 국적)
- **예시**: "미국"

### 3. 코드
- **타입**: string
- **설명**: 병원 식별 코드 (정제주소 + 의원명 조합)
- **예시**: "경상북도 칠곡군 왜관읍 중앙로9길 23_자선의원"

### 4. 의원명
- **타입**: string
- **설명**: 병원 이름
- **예시**: "자선의원"

### 5. 개설일자
- **타입**: date
- **설명**: 병원 개설일
- **예시**: "1988-06-30"

### 6. 종별코드명
- **타입**: string
- **설명**: 병원 종별 (의원, 병원 등)
- **예시**: "의원"

### 7. 상태
- **타입**: string
- **설명**: 병원 영업 상태
- **예시**: "폐업"

### 8. 폐업일자
- **타입**: date, nullable
- **설명**: 폐업일
- **예시**: "2024-03-04"

### 9. 오픈닥터_진료과
- **타입**: string
- **설명**: 주요 진료과
- **예시**: "산부인과"

### 10. 총면적m2
- **타입**: number
- **설명**: 병원 총 면적(m²)
- **예시**: 150

### 11. 정제주소
- **타입**: string
- **설명**: 주소(정규화/정제된 형태)
- **예시**: "경상북도 칠곡군 왜관읍 중앙로9길 23"

### 12. 매출액
- **타입**: number
- **설명**: 해당 YearMonth의 총 매출액 (단위: 원)
- **예시**: 585000

### 13. 거래건수
- **타입**: int
- **설명**: 결제/거래 건수
- **예시**: 26

### 14. 평균결재액
- **타입**: number
- **설명**: 1건당 평균 결제 금액
- **예시**: 22500

### 15. 주중비율
- **타입**: number (%)
- **설명**: 평일 매출 비중(%)
- **예시**: 73.5

### 16. 주말비율
- **타입**: number (%)
- **설명**: 주말 매출 비중(%)
- **예시**: 26.5

### 17. 월요일
- **타입**: number (%)
- **설명**: 월요일 매출 비중(%)
- **예시**: 0

### 18. 화요일
- **타입**: number (%)
- **설명**: 화요일 매출 비중(%)
- **예시**: 24.79

### 19. 수요일
- **타입**: number (%)
- **설명**: 수요일 매출 비중(%)
- **예시**: 10.26

### 20. 목요일
- **타입**: number (%)
- **설명**: 목요일 매출 비중(%)
- **예시**: 7.69

### 21. 금요일
- **타입**: number (%)
- **설명**: 금요일 매출 비중(%)
- **예시**: 10.26

### 22. 토요일
- **타입**: number (%)
- **설명**: 토요일 매출 비중(%)
- **예시**: 20.51

### 23. 일요일
- **타입**: number (%)
- **설명**: 일요일 매출 비중(%)
- **예시**: 26.5

## 데이터 특징
- 7개 요일 비율의 합계는 약 100%
- 주중비율 + 주말비율 = 100%
- 각 병원의 월별 매출 추이를 시계열로 분석 가능
- 국가별로 분리되어 있어 외국인 환자 분석 가능
